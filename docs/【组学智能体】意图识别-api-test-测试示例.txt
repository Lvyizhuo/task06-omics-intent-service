# 组学智能体意图识别服务 - 测试用例

> 注意：实际运行时 DNA 序列会作为参数传入下游 API（PlantCAD2 / EVO2），
> 如果服务未部署或序列无效，参数提取仍会成功，但接口调用会报错。
> 本测试主要验证**意图识别 + 参数提取**是否准确。

## 1. 高置信度测试（应返回 task_id 和 params）

### 1.1 掩码预测 (203)
```bash
curl http://localhost:8010/intent/recognize \
  -H "Content-Type: application/json" \
  -d '{
    "user_input": "请帮我预测DNA序列ACGTACGTACGT在位置1,3,5的碱基概率分布",
    "session_id": "test_001"
  }'
```
